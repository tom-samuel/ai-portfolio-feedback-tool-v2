# AI Portfolio Feedback Tool

**Description:**  
A web application that provides instant AI-powered feedback on designers' portfolios, focusing on UI/UX, usability, and aesthetics.

**Tech Stack:**  
- HTML5 + TailwindCSS  
- Vanilla JavaScript  
- OpenAI GPT API for AI feedback

**Features:**  
- Responsive and modern UI  
- Input validation for portfolio URLs  
- Loading animation during AI response  
- Animated feedback cards for enhanced UX

**How to Use:**  
1. Paste your portfolio URL into the input field.  
2. Click "Get Feedback".  
3. Wait a few seconds for AI to generate detailed design feedback.

**Future Improvements:**  
- User authentication for saving past feedback  
- Multi-portfolio analysis  
- Enhanced styling with dark mode toggle
